# Calendar & Availability Management

Effectively managing your calendar and availability is crucial for maximizing bookings and maintaining professional relationships with clients.

## Overview

The Servineo calendar system allows fixers to:

- View upcoming appointments
- Manage booking requests
- Set availability windows
- Track appointment history
- Handle cancellations

<Info>
Proper calendar management can increase your booking rate by up to 60% by showing clients when you're available.
</Info>

### Appointment Structure

```typescript
interface Appointment {
  _id: string;
  current_requester_name: string;
  current_requester_phone: string;
  appointment_description: string;
  starting_time: Date;
  finishing_time: Date;
  display_name_location?: string;
  lat?: number;
  lon?: number;
  schedule_state: 'booked' | 'pending' | 'completed' | 'cancelled';
  appointment_type: 'virtual' | 'presencial';
  link_id?: string;  // For virtual appointments
  id_requester: string;
}
```

### Converting Appointments to Jobs

Appointments can be converted into formal jobs:

```typescript
const handleCreateJob = async (appointment: Appointment) => {
  const jobData = {
    appointmentId: appointment._id,
    jobMotive: appointment.appointment_description,
    jobDescription: appointment.appointment_description,
    location: {
      lat: appointment.lat,
      lng: appointment.lon,
      direccion: appointment.display_name_location
    },
    scheduledTime: {
      start: appointment.starting_time,
      end: appointment.finishing_time
    }
  };
  
  await createJobFromAppointment(jobData);
};
```

<Tip>
Converting appointments to jobs enables formal payment processing and tracking through the platform.
</Tip>

### Blocking Specific Dates

Block dates when you're unavailable:

```typescript
const blockDates = async (dates: Date[]) => {
  await updateAvailability({
    blockedDates: dates,
    reason: 'vacation' | 'personal' | 'other'
  });
};
```

<Warning>
Blocking dates doesn't automatically cancel existing appointments. You must manually cancel or reschedule them.
</Warning>

### Cancelled Appointments API

Retrieve cancelled appointments:

```typescript
const getCancelledAppointments = async (fixerId: string, date: Date) => {
  const response = await fetch(
    `${API_BASE}/api/crud_read/schedules/get_cancelled_appointments_by_fixer_date`,
    {
      method: 'POST',
      body: JSON.stringify({ fixerId, date })
    }
  );
  return response.json();
};
```

<Note>
Excessive cancellations can negatively impact your profile rating. Try to maintain a cancellation rate below 10%.
</Note>
