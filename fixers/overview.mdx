# Fixer Overview

As a **Fixer** on Servineo, you are a verified service provider who offers professional services to customers in your area. This guide will help you understand the fixer role, capabilities, and how to maximize your earnings on the platform.

## What is a Fixer?

A Fixer is a professional service provider who:

- Offers specialized services (plumbing, electrical work, carpentry, painting, etc.)
- Has completed the registration and verification process
- Can create and manage job offers
- Receives payments through the Servineo platform
- Maintains a professional profile with ratings and reviews

<Info>
Fixers must complete a 7-step onboarding wizard before they can start receiving job requests.
</Info>

## Fixer Profile Structure

Your fixer profile includes several key components:

### Essential Information

```typescript
interface FixerProfile {
  // Identity
  ci: string;                    // National ID number
  url_photo: string;             // Profile photo URL
  
  // Location
  workLocation: {
    lat: number;
    lng: number;
    direccion?: string;
    departamento?: string;
  };
  
  // Services
  servicios: string[];           // Array of service IDs
  
  // Payment Methods
  metodoPago: {
    hasEfectivo: boolean;        // Accepts cash
    qr: boolean;                 // Accepts QR payments
    tarjetaCredito: boolean;     // Accepts credit cards
  };
  
  // Vehicle
  vehiculo: {
    hasVehiculo: boolean;
    tipoVehiculo?: 'motorcycle' | 'car' | 'van' | 'truck';
  };
  
  // Additional
  experience?: {
    descripcion: string;
  };
  accountInfo?: string;          // For digital payments
}
```

### Profile Status

Fixers have different profile statuses:

- **pending**: Initial registration submitted, awaiting verification
- **completed**: Profile verified and active
- **incomplete**: Registration started but not finished

<Warning>
Only fixers with a "completed" status can receive job requests and payments.
</Warning>

## Job Statistics

Track your performance with real-time statistics:

- **Completed Jobs**: Successfully finished work
- **In Progress**: Currently active projects
- **Pending Jobs**: Jobs awaiting confirmation or start
- **Total Jobs**: Overall job count

Statistics are visualized in a pie chart showing the distribution of your work across different states.

## Getting Started

Ready to become a fixer? Follow these steps:

1. **Review Requirements**: Ensure you meet the platform requirements
2. **Complete Registration**: Follow the [Getting Started](/fixers/getting-started) guide
3. **Set Up Your Profile**: Complete all 7 onboarding steps
4. **Create Job Offers**: List your services as described in [Job Offers](/fixers/job-offers)
5. **Start Earning**: Begin receiving and completing jobs
