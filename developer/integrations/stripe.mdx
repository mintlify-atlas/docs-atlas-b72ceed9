# Stripe Payment Integration

Servineo integrates with Stripe to process credit card payments securely. This integration supports card payments, payment method storage, and payment intent creation.

## Overview

The Stripe integration uses official Stripe libraries to handle payment processing:

- `@stripe/stripe-js` - Load Stripe.js for secure payment form rendering
- `@stripe/react-stripe-js` - React components for Stripe Elements

**Packages:**
- `@stripe/stripe-js` v8.5.3
- `@stripe/react-stripe-js` v5.4.1

## Setup

<Steps>
  <Step title="Install Dependencies">
    The required packages are already included:
    
    ```bash
    npm install @stripe/stripe-js @stripe/react-stripe-js
    ```
  </Step>

  <Step title="Configure Environment Variables">
    Add your Stripe publishable key to your environment:
    
    ```bash .env.local
    NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_your_key_here
    NEXT_PUBLIC_BACKEND_URL=https://your-api-domain.com
    ```
    
    <Warning>
      Never commit your Stripe secret key to version control. Store it securely on your backend server.
    </Warning>
  </Step>

  <Step title="Initialize Stripe">
    Load Stripe with your publishable key at the application level.
  </Step>
</Steps>

### Add Card Modal Component

Implement a component to add and save payment methods:

```tsx src/app/[locale]/payment/components/AddCardModal.tsx
'use client';
import { useState } from 'react';
import { CardElement, useStripe, useElements } from '@stripe/react-stripe-js';

interface AddCardModalProps {
  userId: string;
  fixerId: string;
  jobId: string;
  amount: number;
  onClose: () => void;
  onCardAdded: (payload: any) => void;
}

export default function AddCardModal({
  userId,
  fixerId,
  jobId,
  amount,
  onClose,
  onCardAdded
}: AddCardModalProps) {
  const stripe = useStripe();
  const elements = useElements();
  const [saveCard, setSaveCard] = useState(false);
  const [loading, setLoading] = useState(false);
  const [errorMessage, setErrorMessage] = useState('');
  const [cardHolder, setCardHolder] = useState('');

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    
    if (!stripe || !elements) {
      setErrorMessage('Stripe aún no está listo, espera unos segundos.');
      return;
    }

    setLoading(true);
    setErrorMessage('');
    
    const cardElement = elements.getElement(CardElement);
    if (!cardElement) {
      setErrorMessage('No se pudo obtener la tarjeta.');
      setLoading(false);
      return;
    }

    try {
      // Create payment method
      const { error, paymentMethod } = await stripe.createPaymentMethod({
        type: 'card',
        card: cardElement,
        billing_details: { name: cardHolder },
      });
      
      if (error) throw new Error(error.message);

      let cardId = null;
      
      // Save card if requested
      if (saveCard) {
        const cardRes = await fetch(
          `${process.env.NEXT_PUBLIC_BACKEND_URL}/api/cardscreate`,
          {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
              userId,
              paymentMethodId: paymentMethod.id,
              saveCard,
              cardholderName: cardHolder,
            }),
          }
        );
        
        const savedCard = await cardRes.json();
        if (!cardRes.ok) throw new Error(savedCard.error || 'Error al guardar la tarjeta');
        cardId = savedCard._id;
      }

      // Create payment
      const paymentRes = await fetch(
        `${process.env.NEXT_PUBLIC_BACKEND_URL}/api/createpayment`,
        {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({
            requesterId: userId,
            fixerId,
            jobId,
            cardId: cardId || null,
            paymentMethodId: saveCard ? undefined : paymentMethod.id,
            amount,
          }),
        }
      );
      
      const paymentData = await paymentRes.json();
      if (!paymentRes.ok) throw new Error(paymentData.error || 'Error al crear el pago');

      onCardAdded({
        payment: {
          _id: paymentData.payment._id,
          amount: paymentData.payment.amount,
          card: cardId ? { _id: cardId } : null,
        },
        cardSaved: saveCard,
      });

      cardElement.clear();
      setCardHolder('');
      
      setTimeout(() => {
        onClose();
      }, 2000);

    } catch (err: unknown) {
      console.error(err);
      if (err instanceof Error) {
        setErrorMessage(err.message);
      } else {
        setErrorMessage('Ocurrió un error inesperado.');
      }
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="fixed inset-0 bg-black/60 flex justify-center items-center z-50">
      <div className="bg-white p-6 rounded-2xl shadow-2xl w-96">
        <h2 className="text-xl font-bold mb-3 text-center">
          Agregar nueva tarjeta
        </h2>
        
        <p className="text-center mb-4">
          Monto a pagar: <strong>{amount} BOB</strong>
        </p>

        {errorMessage && (
          <div className="bg-red-100 border border-red-500 text-red-700 p-2 mb-4 rounded">
            {errorMessage}
          </div>
        )}

        <form onSubmit={handleSubmit} className="space-y-4">
          <div>
            <label className="block mb-2 text-sm font-semibold">
              Nombre de titular*
            </label>
            <input
              type="text"
              value={cardHolder}
              onChange={(e) => setCardHolder(e.target.value)}
              placeholder="Ej: Juan Pérez"
              className="w-full border rounded-xl px-3 py-2"
              required
            />
          </div>

          <div className="p-3 rounded-xl border">
            <CardElement
              options={{
                hidePostalCode: true,
                style: {
                  base: {
                    fontSize: '16px',
                    color: '#000',
                    '::placeholder': { color: '#888' },
                  },
                  invalid: { color: '#ff6b6b' },
                },
              }}
            />
          </div>

          <label className="flex items-center gap-2 text-sm">
            <input
              type="checkbox"
              checked={saveCard}
              onChange={(e) => setSaveCard(e.target.checked)}
              className="accent-blue-500"
            />
            Guardar tarjeta para futuros pagos
          </label>

          <div className="flex justify-between mt-6">
            <button
              type="button"
              onClick={onClose}
              className="px-4 py-2 bg-gray-200 hover:bg-gray-300 rounded-xl"
            >
              Cancelar
            </button>
            <button
              type="submit"
              disabled={!stripe || loading}
              className={`px-5 py-2 rounded-xl font-semibold ${
                !stripe || loading
                  ? 'bg-gray-300 cursor-not-allowed'
                  : 'bg-blue-500 hover:bg-blue-600 text-white'
              }`}
            >
              {loading ? 'Procesando...' : 'Pagar'}
            </button>
          </div>
        </form>
      </div>
    </div>
  );
}
```

## Payment Flow

<Steps>
  <Step title="Initialize Payment Intent">
    Create a payment intent on your backend and return the client secret.
  </Step>

  <Step title="Collect Card Details">
    Use `CardElement` to securely collect card information. Stripe handles PCI compliance.
  </Step>

  <Step title="Create Payment Method">
    ```typescript
    const { error, paymentMethod } = await stripe.createPaymentMethod({
      type: 'card',
      card: cardElement,
      billing_details: { name: cardHolder },
    });
    ```
  </Step>

  <Step title="Send to Backend">
    Send the payment method ID to your backend to complete the payment.
    
    ```typescript
    const response = await fetch('/api/createpayment', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        paymentMethodId: paymentMethod.id,
        amount,
        jobId,
      }),
    });
    ```
  </Step>

  <Step title="Confirm Payment">
    Your backend confirms the payment with Stripe and returns the result.
  </Step>
</Steps>

## Error Handling

<Warning>
  Always handle Stripe errors gracefully and provide clear user feedback.
</Warning>

```typescript
try {
  const { error, paymentMethod } = await stripe.createPaymentMethod({
    type: 'card',
    card: cardElement,
  });
  
  if (error) {
    throw new Error(error.message);
  }
  
  // Process payment
} catch (err) {
  if (err instanceof Error) {
    setErrorMessage(err.message);
  } else {
    setErrorMessage('An unexpected error occurred.');
  }
}
```

## Common Stripe Errors

| Error Code | Description | Solution |
|------------|-------------|----------|
| `card_declined` | Card was declined | Ask user to use different card |
| `expired_card` | Card has expired | Request updated card details |
| `incorrect_cvc` | CVC is incorrect | Verify CVC number |
| `insufficient_funds` | Insufficient funds | Use different payment method |
| `processing_error` | Error processing card | Retry or contact support |

## Security Best Practices

<Info>
  - Never handle raw card data on your server
  - Use Stripe Elements for PCI-compliant card input
  - Always validate on the backend
  - Use HTTPS in production
  - Implement 3D Secure for additional security
</Info>

## Backend Integration

Your backend should handle:

1. **Create Payment Intent**
```javascript
const paymentIntent = await stripe.paymentIntents.create({
  amount: amount * 100, // Convert to cents
  currency: 'bob',
  payment_method: paymentMethodId,
  confirm: true,
});
```

2. **Save Payment Method**
```javascript
const paymentMethod = await stripe.paymentMethods.attach(
  paymentMethodId,
  { customer: customerId }
);
```

3. **Handle Webhooks**
```javascript
app.post('/webhook', async (req, res) => {
  const sig = req.headers['stripe-signature'];
  const event = stripe.webhooks.constructEvent(req.body, sig, webhookSecret);
  
  if (event.type === 'payment_intent.succeeded') {
    // Handle successful payment
  }
});
```
