Welcome to Servineo! This guide will walk you through creating an account and completing your first service request or offer. Whether you need home repairs or want to offer professional services, you'll be up and running in no time.

## What is Servineo?

Servineo is a platform that connects service requesters with professional fixers for home repairs and maintenance. The platform supports two user roles:

- **Requesters**: Find and hire professionals for home services
- **Fixers**: Offer professional services and connect with clients

<Note>
  The platform uses role-based access, so your experience will be tailored based on whether you're a requester or a fixer.
</Note>

## Getting Started as a Requester

<Steps>
  <Step title="Create Your Account">
    Navigate to the sign-up page and create your account. You can register using:
    
    - Email and password
    - Google authentication
    - GitHub authentication
    - Discord authentication
    
    ```typescript
    // User interface structure
    interface IUser {
      name: string;
      email: string;
      role: 'requester' | 'fixer' | 'admin';
      telefono?: string;
      ubicacion?: {
        lat?: number;
        lng?: number;
        direccion?: string;
      };
    }
    ```
    
    <Info>
      You'll need to complete ReCAPTCHA verification during registration for security.
    </Info>
  </Step>

  <Step title="Complete Your Profile">
    After signing up, add your:
    - Phone number (required for service coordination)
    - Location information (helps match you with nearby fixers)
    - Profile photo (optional, builds trust)
    
    Your location is critical for finding the best fixers in your area and getting accurate service quotes.
  </Step>

  <Step title="Browse Available Services">
    Explore the services offered on the platform. Servineo supports a wide range of home services:
    
    <CardGroup cols={3}>
      <Card title="Plumbing" icon="pipe">
        Water systems, repairs, installations
      </Card>
      <Card title="Electrical" icon="bolt">
        Wiring, installations, repairs
      </Card>
      <Card title="Carpentry" icon="hammer">
        Furniture, installations, repairs
      </Card>
      <Card title="Painting" icon="paintbrush">
        Interior and exterior painting
      </Card>
      <Card title="Gardening" icon="leaf">
        Landscaping, maintenance, design
      </Card>
      <Card title="Cleaning" icon="broom">
        Home and office cleaning services
      </Card>
    </CardGroup>
    
    You can search for fixers by:
    - Service type
    - Location (using the interactive map)
    - Rating and reviews
    - Price range
  </Step>

  <Step title="Submit Your First Service Request">
    When you find a fixer or service you need:
    
    1. Click **"Request Service"** on the fixer's profile or service offer
    2. Fill out the job request form:
    
    ```typescript
    interface CreateJobRequestPayload {
      title: string;              // Brief description of work
      description: string;         // Detailed requirements
      location: {
        type: 'Point';
        coordinates: [string, string];  // [longitude, latitude]
      };
      startTime: string;          // When work should begin
      endTime: string;            // Estimated completion
      price: string | number;     // Your budget or agreed price
      fixerId: string;            // Selected professional
      requesterId: string;        // Your user ID
    }
    ```
    
    3. Specify the job details:
       - **Job Motive**: What needs to be fixed or installed
       - **Description**: Provide details about the work
       - **Location**: Confirm or modify your location
       - **Time Window**: Start and end times
       - **Suggested Rate**: Your budget for the job
    
    4. Submit the request
    
    <Warning>
      Make sure your location and time availability are accurate. Fixers rely on this information to accept and schedule jobs.
    </Warning>
  </Step>

  <Step title="Track Your Request">
    After submitting, you can:
    - View request status (pending, accepted, in_progress, completed)
    - Communicate with your fixer
    - Track appointments in your calendar
    - Manage payments through the platform
    
    Job statuses include:
    - `pending`: Waiting for fixer acceptance
    - `accepted`: Fixer has confirmed
    - `in_progress`: Work is underway
    - `completed`: Job finished
    - `paid`: Payment processed
  </Step>

  <Step title="Rate Your Experience">
    After job completion:
    - Leave a rating (1-5 stars)
    - Write a review
    - Help other users make informed decisions
    
    Your feedback helps maintain quality standards across the platform.
  </Step>
</Steps>

## Getting Started as a Fixer

<Steps>
  <Step title="Sign Up as a Requester First">
    All fixers start as requesters. Create your basic account following the steps above.
    
    <Note>
      The platform uses a conversion process to upgrade requester accounts to fixer status.
    </Note>
  </Step>

  <Step title="Convert to Fixer Account">
    Navigate to the **"Become a Fixer"** page (`/become-fixer`) and complete the fixer registration wizard:
    
    ```typescript
    // Fixer registration form fields
    interface InitialRegistrationData {
      name: string;        // Full name (max 30 chars, letters only)
      email: string;       // Contact email
      phone: string;       // Phone number with country code
    }
    ```
    
    The system validates:
    - Name format (letters and spaces only, max 30 characters)
    - Valid email address
    - Phone number format
  </Step>

  <Step title="Complete Fixer Profile">
    Add your professional information:
    - **Services**: Select the services you offer (plumbing, electrical, etc.)
    - **Work Location**: Set your service area with coordinates
    - **Experience**: Describe your professional background
    - **Vehicle**: Specify if you have transportation
    - **Payment Methods**: Cash, QR codes, credit cards
    - **CI/ID**: Identification for verification
    
    ```typescript
    interface IUser {
      servicios?: string[];          // Services offered
      workLocation?: {
        lat?: number;
        lng?: number;
        direccion?: string;
      };
      experience?: {
        descripcion?: string;
      };
      vehiculo?: {
        hasVehiculo?: boolean;
        tipoVehiculo?: string;
      };
      metodoPago?: {
        hasEfectivo?: boolean;
        qr?: boolean;
        tarjetaCredito?: boolean;
      };
    }
    ```
  </Step>

  <Step title="Create Your First Job Offer">
    Post services you want to offer:
    
    1. Go to your fixer dashboard
    2. Click **"Create Job Offer"**
    3. Fill in the details:
       - Service type
       - Description
       - Pricing
       - Availability
       - Service area
    
    ```typescript
    interface IJobOffer {
      title: string;
      description: string;
      price: number;
      fixerId: string;
      serviceType: string;
      isActive: boolean;  // Toggle visibility
    }
    ```
  </Step>

  <Step title="Manage Incoming Requests">
    When requesters find you:
    - Review job requests in your dashboard
    - Accept or decline based on your availability
    - Communicate job details
    - Schedule appointments
    - Track job progress
  </Step>

  <Step title="Get Paid">
    After completing jobs:
    - Use the payment center (`/payment/centro-de-pagos`)
    - Access your fixer wallet (`/payment/FixerWallet`)
    - Track payment history
    - Manage invoices
    - Add bank account details for transfers
  </Step>
</Steps>

## Authentication & Security

Servineo uses secure authentication with multiple options:

```typescript
// Authentication providers supported
interface AuthProviders {
  provider: 'email' | 'google' | 'github' | 'discord';
  providerId: string;
  password?: string;  // Only for email auth
}
```

<Info>
  All authentication flows are protected with ReCAPTCHA to prevent automated abuse.
</Info>

### Security Features

- Two-factor authentication available (`/requesterEdit/Seguridad/Authenticator`)
- Secure session management
- Password reset flow (`/login/forgotpass`)
- Account linking for multiple providers
