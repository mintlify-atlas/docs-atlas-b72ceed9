# Digital Wallet & Payments

Your Servineo digital wallet is the central hub for managing payments, tracking earnings, and withdrawing funds from completed jobs.

## Wallet Overview

The Fixer Wallet provides:

- Real-time balance tracking
- Transaction history
- Payment processing
- Bank account management
- Earnings analytics
- Withdrawal functionality

<Info>
Access your wallet at `/payment/FixerWallet?fixerId={YOUR_FIXER_ID}`
</Info>

### Payment Center Data Structure

```typescript
interface FixerData {
  saldoActual: number;          // Current balance
  totalGanado: number;          // Total earned
  trabajosCompletados: number;  // Completed jobs count
  fixerId: string;
  isTestData?: boolean;
  
  lowBalanceAlert?: {
    type: 'none' | 'low' | 'critical';
  };
  
  lowBalanceInfo?: {
    balance: number;
    currency: string;
    lowBalanceThreshold: number;
    lastLowBalanceNotification?: Date;
    
    flags?: {
      needsLowAlert?: boolean;
      needsCriticalAlert?: boolean;
      cooldownUntil?: Date | null;
    };
    
    toast?: {
      shouldShow: boolean;
      level: 'low' | 'critical' | 'none';
      message: string;
    };
  };
}
```

## Low Balance Alerts

The system monitors your balance and sends alerts:

### Alert Levels

<CardGroup cols={3}>
  <Card title="Normal" icon="check-circle">
    Balance above threshold - no alerts
  </Card>
  
  <Card title="Low" icon="exclamation-triangle">
    Balance below threshold - warning toast
  </Card>
  
  <Card title="Critical" icon="exclamation-circle">
    Negative or critically low - urgent toast
  </Card>
</CardGroup>

### Alert Toast Display

```typescript
useEffect(() => {
  const toastInfo = fixerData?.lowBalanceInfo?.toast;
  
  if (toastInfo?.shouldShow) {
    if (toastInfo.level === 'critical') {
      showToast.error(toastInfo.message, {
        duration: 8000,
        position: 'top-center'
      });
    } else if (toastInfo.level === 'low') {
      showToast.warning(toastInfo.message, {
        duration: 8000,
        position: 'top-center'
      });
    }
  }
}, [fixerData?.lowBalanceInfo?.toast]);
```

<Warning>
Maintain a positive balance to continue receiving job requests. Low balance may limit your visibility on the platform.
</Warning>

### Cash Payment Flow

```typescript
// Create cash payment
const createCashPayment = async (dto: CreateCashPaymentDTO) => {
  const response = await fetch('/api/payments/cash', {
    method: 'POST',
    body: JSON.stringify({
      jobId: dto.jobId,
      amount: dto.amount,
      fixerId: dto.fixerId,
      paymentMethods: 'cash'
    })
  });
  
  return response.json();
};
```

### Confirming Cash Payments

For cash payments, you must confirm receipt:

<Steps>
  <Step title="Client Generates Code">
    Client creates a payment code in their app
  </Step>
  
  <Step title="Collect Cash">
    Receive cash payment from client
  </Step>
  
  <Step title="Enter Code">
    Input the payment code in your app
  </Step>
  
  <Step title="Confirm Payment">
    Click "Confirmar pago" to complete transaction
  </Step>
</Steps>

```typescript
const confirmPayment = async (paymentId: string) => {
  const response = await fetch(`/api/lab/payments/${paymentId}/confirm`, {
    method: 'POST'
  });
  
  if (response.ok) {
    // Payment confirmed, funds added to wallet
  }
};
```

<Warning>
Never confirm a cash payment before actually receiving the money. Payment confirmations are irreversible.
</Warning>

### QR Payments

QR payments are processed automatically:

1. Client scans your QR code
2. Payment processes through payment gateway
3. Funds deposit to your wallet automatically
4. You receive notification of payment

## Withdrawals

Withdraw funds from your wallet to your bank account:

### Withdrawal Requirements

- Minimum withdrawal: Bs. 50.00
- Verified bank account required
- Positive balance above minimum
- No pending disputes

### Withdrawal Process

<Steps>
  <Step title="Request Withdrawal">
    Enter amount to withdraw from wallet
  </Step>
  
  <Step title="Verify Details">
    Confirm bank account and amount
  </Step>
  
  <Step title="Process">
    Withdrawal request submitted for processing
  </Step>
  
  <Step title="Transfer">
    Funds transfer to bank (1-3 business days)
  </Step>
</Steps>

```typescript
const requestWithdrawal = async (amount: number) => {
  const response = await fetch('/api/payments/withdraw', {
    method: 'POST',
    body: JSON.stringify({
      fixerId,
      amount,
      bankAccountId
    })
  });
  
  return response.json();
};
```

<Note>
Withdrawals are processed on business days only. Requests submitted on weekends or holidays process on the next business day.
</Note>

## Transaction History

### Transaction Types

## Troubleshooting

<AccordionGroup>
  <Accordion title="Balance not updating">
    Payment processing can take 5-10 minutes. If balance doesn't update after 1 hour, check transaction history or contact support.
  </Accordion>
  
  <Accordion title="Can't link bank account">
    Ensure all bank details are correct. Account number must match bank records exactly.
  </Accordion>
  
  <Accordion title="Withdrawal failed">
    Check that you have sufficient balance and a verified bank account. Ensure account is active and accepting deposits.
  </Accordion>
  
  <Accordion title="Missing payment">
    Verify with client that payment was sent. Check transaction history for pending payments.
  </Accordion>
  
  <Accordion title="Low balance alert not clearing">
    Recharge your wallet above the threshold amount (typically Bs. 100). Alert clears within 15 minutes.
  </Accordion>
</AccordionGroup>
