# Job Offers Management

Job offers are your service listings on Servineo. This guide covers everything you need to know about creating and managing effective job offers.

## What are Job Offers?

Job offers are service listings that:

- Showcase your services to potential clients
- Include pricing, descriptions, and photos
- Can be toggled active/inactive
- Support promotional campaigns
- Display in search results and fixer profiles

<Info>
Active job offers appear in search results and on your public profile. Inactive offers are hidden from clients but remain in your dashboard.
</Info>

## Job Offer Structure

```typescript
interface IJobOffer {
  _id: string;
  fixerId: string;
  fixerName: string;
  title: string;
  description: string;
  category: string;
  tags: string[];
  price: number;
  city: string;
  contactPhone: string;
  photos: string[];
  rating: number;
  status: boolean;  // true = active, false = inactive
  createdAt: Date;
}
```

The job offer form uses Zod schema validation:

## Photo Management

### Upload Requirements

- **Format**: JPG, PNG, WebP
- **Size**: Maximum 5MB per image
- **Count**: 1-5 photos per offer
- **Recommended**: High-quality, well-lit photos

### Photo Upload Process

```tsx
const handleImageChange = (e: React.ChangeEvent<HTMLInputElement>) => {
  const files = Array.from(e.target.files || []);
  
  // Validate count
  if (selectedImages.length + files.length > 5) {
    showNotify('warning', 'Máximo 5 imágenes');
    return;
  }
  
  // Validate size
  files.forEach((file) => {
    if (file.size > 5 * 1024 * 1024) {
      showNotify('error', 'Imagen muy grande');
    }
  });
};
```

<Warning>
You must upload at least 1 photo when creating a new offer. Editing existing offers does not require new photos.
</Warning>

### Photo Preview

Before submission:

- Click X to delete individual photos

### Adding Tags

### Tag Rules

- Minimum: 1 tag (automatically includes category)
- Maximum: 5 tags
- No duplicate tags
- Tags can be removed by clicking the X icon

```tsx
const handleAddTag = (value: string) => {
  if (!value) return;
  
  if (!currentTags.includes(value) && currentTags.length < 5) {
    setValue('tags', [...currentTags, value]);
  }
};
```

<Tip>
Use specific tags that clients might search for. For example, a plumbing offer could include: "Plomería", "Reparaciones", "Emergencias", "Instalación".
</Tip>

## Offer Status Management

### Active vs Inactive

<CardGroup cols={2}>
  <Card title="Active Offers" icon="check-circle">
    Visible to clients in search results and your profile
  </Card>
  
  <Card title="Inactive Offers" icon="x-circle">
    Hidden from clients, kept in your dashboard for later reactivation
  </Card>
</CardGroup>

### Toggling Status

```typescript
const handleToggleActive = async (jobId: string) => {
  await toggleJobStatus({ jobId }).unwrap();
  await refetch(); // Refresh list
};
```

Offers can be toggled active/inactive:

- Click the toggle button on the offer card
- Status changes immediately
- Inactive offers show with dimmed styling

<Info>
Use inactive status when you're temporarily unable to take new clients, rather than deleting the offer.
</Info>

## Filtering Offers

Filter your offers by status:

```tsx
<select onChange={(e) => setStatusFilter(e.target.value)}>
  <option value="all">Todas</option>
  <option value="active">Activas</option>
  <option value="inactive">Inactivas</option>
</select>
```

**Filter Options:**
- **All**: Show all offers regardless of status
- **Active**: Show only active offers
- **Inactive**: Show only inactive offers

## Deleting Job Offers

<Warning>
Deleting an offer is permanent and cannot be undone. Consider using inactive status instead.
</Warning>

const confirmDelete = (jobId: string) => {
  showNotify('warning', 'Confirmar eliminación', () => {

## Troubleshooting

<AccordionGroup>
  <Accordion title="Photos not uploading">
    Check file size (max 5MB) and format (JPG, PNG, WebP). Clear browser cache if issues persist.
  </Accordion>
  
  <Accordion title="Can't submit offer">
    Ensure all required fields are filled and at least 1 photo is uploaded for new offers.
  </Accordion>
  
  <Accordion title="Tags not saving">
    You must have at least 1 tag and maximum 5 tags. Category is automatically included.
  </Accordion>
  
  <Accordion title="Offer not appearing in search">
    Check that offer status is "active". Inactive offers don't appear in client searches.
  </Accordion>
</AccordionGroup>

<Note>
Changes to job offers typically reflect in search results within 5-10 minutes due to caching.
</Note>
