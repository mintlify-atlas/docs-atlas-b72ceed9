Servineo offers a comprehensive suite of features designed to streamline the process of requesting and offering home services. This guide covers all major platform capabilities.

## Core Features

### Role-Based Experience

Servineo provides customized experiences based on user roles:

```typescript
interface IUser {
  role: 'requester' | 'fixer' | 'admin';
}
```

<Info>
  The platform automatically adjusts the interface, navigation, and available features based on your role.
</Info>

### 2. Job Request Management

#### Creating Job Requests

Submit detailed service requests with:

```typescript
interface CreateJobRequestPayload {
  title: string;                    // Job summary
```

#### Location Updates

<CodeGroup>
```typescript API Call
// Update user location
const updateLocation = async (userId: string, location: {
  lat: number;
  lng: number;
  direccion?: string;
}) => {
  const response = await fetch(`/api/user/${userId}/location`, {
    method: 'PATCH',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ workLocation: location })
  });
  return response.json();
};
```

```typescript Redux Hook
import { useUpdateLocationMutation } from '@/app/redux/services/userApi';

const [updateLocation] = useUpdateLocationMutation();

await updateLocation({
  id: userId,

});
```
