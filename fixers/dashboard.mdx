# Fixer Dashboard

The fixer dashboard is your command center for managing your profile, job offers, and business performance on Servineo.

## Dashboard Overview

Access your dashboard at `/fixer/dashboard` after completing registration. The dashboard consists of:

- **Profile Header**: Your profile card with key information
- **Sidebar Navigation**: Quick access to all dashboard sections
- **Main Content Area**: Displays the selected section content

## Profile Header

The profile header displays your essential information:

<CardGroup cols={2}>
  <Card title="Profile Photo" icon="user-circle">
    Your profile picture with completion status indicator
  </Card>
  
  <Card title="Profile Information" icon="id-card">
    Name, description, contact details, and location
  </Card>
  
  <Card title="Services & Skills" icon="wrench">
    List of services you offer as skill tags

  <Card title="Quick Actions" icon="bolt">

### Profile Status Indicator

```tsx
{user.fixerProfile === 'completed' && (
  <div className="badge-completed" title="Perfil Completo">
    ✓
  </div>
)}
```

A green checkmark badge appears when your profile is fully completed and verified.

### Editable Description

Your profile description is prominently displayed and editable:

<Steps>
  <Step title="Click Edit Profile">
    The description field becomes editable with a textarea
  </Step>
  
  <Step title="Write Description">
    Describe your experience and services (10-255 characters)
  </Step>
  
  <Step title="Save Changes">
    Click "Guardar" to update your profile
  </Step>
</Steps>

**Description Validation:**
- Minimum: 10 characters
- Maximum: 255 characters
- Character counter displayed
- Real-time validation with Zod schema

<Tip>
A compelling description increases client trust. Highlight your expertise, years of experience, and what makes your service unique.
</Tip>

### Contact Information Display

Your contact details are shown with icons:

- **Email**: `user.email`
- **Phone**: `user.telefono`
- **Location**: `user.ubicacion.departamento` or `user.workLocation.direccion`
- **CI**: National ID (displayed only to you)

### Services Tags

Services are displayed as colored badges:

```tsx
{user.servicios.map((servicio, index) => (
  <span className="service-badge" key={index}>
    {servicio}
  </span>
))}
```

## Dashboard Sections

### 1. Job Offers Section

Manage your service listings. See [Job Offers](/fixers/job-offers) for detailed documentation.

**Key Features:**
- Create new job offers
- Edit existing offers
- Toggle active/inactive status
- Delete offers
- Create promotions
- Filter by status

### 2. Certifications Section

Upload and display professional certifications:

```tsx
<CertificationsSection fixerId={user._id || user.id} />
```

**Functionality:**
- Upload certification documents
- Display certification images
- Add certification titles and descriptions
- Remove outdated certifications

<Note>
Certifications build trust with clients. Upload licenses, training certificates, and professional credentials.
</Note>

### 3. Experience Section

Showcase your professional background:

```tsx
<ExperienceSection fixerId={user._id || user.id} />
```

**What to Include:**
- Years of experience
- Previous companies/employers
- Notable projects
- Special skills or training
- Professional achievements

### 4. Portfolio Section

Display photos of your completed work:

```tsx
<PortfolioSection fixerId={user._id || user.id} />
```

**Portfolio Features:**
- Upload project photos
- Organize by category
- Add captions and descriptions
- Delete photos

<Tip>
Before/after photos are extremely effective. Show the transformation your work provides.
</Tip>

### 5. Location Section

Update your work location:

```tsx
<LocationSection
  fixerId={user._id || user.id}
  currentLocation={{
    lat: user.workLocation?.lat ?? 0,
    lng: user.workLocation?.lng ?? 0,
    direccion: user.workLocation?.direccion || ''
  }}
/>
```

**Location Management:**
- Interactive map interface
- Drag marker to adjust position
- Update street address
- Save new location

<Warning>
Your location affects which job requests you receive. Keep it updated if you move or change service areas.
</Warning>

### 6. Statistics Section

View comprehensive job analytics:

```tsx
<EstadisticasTrabajos />
```

**Metrics Displayed:**

<CardGroup cols={3}>
  <Card title="Completed" icon="check-circle">
    Successfully finished jobs
  </Card>
  
  <Card title="In Progress" icon="clock">
    Currently active jobs
  </Card>
  
  <Card title="Pending" icon="hourglass">
    Jobs awaiting start
  </Card>
</CardGroup>

#### Statistics Visualization

Data is shown in:

- **Pie Chart**: Visual distribution of job states
- **Summary Table**: Detailed counts with percentages
- **Total Count**: Overall job completion metric

```typescript
interface JobStatistics {
  completed: number;
  inProgress: number;
  pending: number;
  total: number;
}
```

## Dashboard State Management

The dashboard uses Redux for state:

```typescript
const { user: reduxUser } = useAppSelector((state) => state.user);
```

**State Updates:**
- Profile edits update Redux store
- Changes sync with backend API
- Real-time updates across dashboard sections

## Performance Optimization

- Lazy loading for images in portfolio
- Paginated job offers list
- Debounced search and filter operations
- Cached statistics data

<Note>
The dashboard automatically refreshes data every 30 seconds to show real-time updates for incoming job requests.
</Note>
